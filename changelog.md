# Changelog du corpus

## 2026-07-16
- Migration vers le site custom. Le corpus est désormais généré depuis le
  miroir Markdown du site et poussé à chaque publication.
- Réorganisation par langue : `articles-md/` et `landing-md/` deviennent
  `fr/blogue/`, `fr/pages/` et `en/blog/`.
- Les URLs gagnent le préfixe de langue : `/blogue/<slug>/` devient
  `/fr/blogue/<slug>/`. Les anciennes URLs redirigent en 301.
- Les noms de fichiers reprennent le slug d'URL, qui ne faisait foi jusqu'ici
  que dans le `canonical_url`.
