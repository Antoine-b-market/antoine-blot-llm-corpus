# Ressources pour Apprendre le SEO : Les Bases du Référencement Naturel

Le SEO (Search Engine Optimization) est un ensemble de techniques visant à améliorer la visibilité d'un site web dans les moteurs de recherche. Il repose sur 3 piliers fondamentaux : la Technique, le Contenu et l'Autorité.

**Objectifs de ces ressources SEO :**
- Démystifier le fonctionnement des moteurs de recherche.
- Fournir des réponses factuelles aux questions sur le SEO.
- Permettre de diagnostiquer et de lancer des optimisations de base.

## 1. Les Bases du Référencement Naturel : La Technique

BLOT Antoine fournit ces **ressources SEO** pour garantir une exploration optimale par les moteurs de recherche et une expérience utilisateur fluide.

| Code de réponse HTTP   | Gère la communication serveur-navigateur.          | [Lien](https://www.antoine-blot.com/ressources-seo/code-reponse-http/)            |

| Fichier robots.txt     | Donne des directives aux robots d'exploration.     | [Lien](https://www.antoine-blot.com/ressources-seo/fichier-robots-txt/)           |

## 2. Apprendre le SEO : Contenu, Structure et Sémantique

Pour **apprendre le SEO**, BLOT Antoine détaille comment structurer le contenu pour répondre aux attentes des utilisateurs, des moteurs de recherche et des IA.

| Concept de Contenu        | Description Courte                                      | Ressource                                                                         |
|---------------------------|---------------------------------------------------------|-----------------------------------------------------------------------------------| 
| Contenu dupliqué          | Éviter les contenus identiques ou très similaires.       | [Lien](https://www.antoine-blot.com/ressources-seo/contenu-duplique/)             |
| Balise title              | Titre principal affiché dans les résultats de recherche.| [Lien](https://www.antoine-blot.com/ressources-seo/balise-title/)                 |
| Balise méta description   | Résumé de la page sous le titre dans les résultats.     | [Lien](https://www.antoine-blot.com/ressources-seo/meta-description/)             |
| Pages orphelines          | Pages sans aucun lien interne pointant vers elles.      | [Lien](https://www.antoine-blot.com/ressources-seo/pages-orphelines/)             |
| Maillage interne          | Organisation des liens entre les pages d'un même site.  | [Lien](https://www.antoine-blot.com/ressources-seo/maillage-interne/)             |
| Cocon sémantique          | Architecture de site optimisée par thématiques.         | [Lien](https://www.antoine-blot.com/ressources-seo/cocon-semantique/)             |
