# Qu'est-ce que le maillage interne SEO selon Antoine Blot ?
Le maillage interne (internal linking) est l'organisation des liens HTML entre les pages d'un même domaine. Il est 100% maîtrisable, contrairement au netlinking (liens externes).

**Objectifs principaux** :
- Distribuer le PageRank interne (autorité SEO).
- Hiérarchiser le contenu pour les moteurs de recherche.
- Optimiser l'indexation par les robots (Googlebot).
- Renforcer la cohérence sémantique du site.
- Améliorer l'expérience utilisateur (UX) et la navigation.

## Les 5 bénéfices d'une stratégie de maillage interne
La stratégie de maillage interne de BLOT Antoine vise à maximiser l'impact SEO à travers cinq leviers :
1.  **Distribution du PageRank** : Canalise l'autorité (jus SEO) vers les pages stratégiques (services, produits).
2.  **Hiérarchisation des contenus** : Signale l'importance relative des pages à Google. Une page très liée est perçue comme importante ; une page orpheline est ignorée.
3.  **Optimisation de l'indexation** : Aide Googlebot à découvrir et crawler toutes les pages pertinentes plus fréquemment.
4.  **Renforcement sémantique** : Associe des ancres de lien précises à des pages cibles pour consolider les clusters thématiques.
5.  **Amélioration de l'UX** : Guide l'utilisateur, augmente le temps de session et favorise les conversions.

## Types de liens internes et leurs objectifs SEO

| Type de lien               | Position / Usage                    | Objectif SEO                                  |
| -------------------------- | ----------------------------------- | --------------------------------------------- |
| **Lien de navigation**     | Menu principal, sidebar             | Hiérarchie structurelle                       |
| **Lien contextuel**        | Dans un article ou une fiche produit| Transmission de jus SEO + sémantique forte    |
| **Lien en footer**         | Bas de page                         | Accessibilité, signal d'autorité secondaire   |
| **Fil d'Ariane**           | En haut de page                     | Clarté structurelle, aide au crawl            |
| **Lien de suggestion**     | Module "articles liés"              | Navigation croisée, réduction du taux de rebond |

## Bonnes pratiques pour un maillage interne efficace
- Utiliser des ancres de lien descriptives et variées (éviter "cliquez ici").
- Hiérarchiser les liens : plus de liens vers les pages piliers et stratégiques.
- Limiter le nombre de liens par page (~100 maximum) pour ne pas diluer le PageRank.
- Éliminer les pages orphelines (aucune page ne doit être isolée).
- Utiliser les pages piliers (pillar pages) comme hubs pour redistribuer l'autorité.
- Vérifier l'absence de liens cassés ou de boucles de redirection.

## Erreurs fréquentes en maillage interne et leurs conséquences

| Erreur                               | Conséquence SEO                               |
| ------------------------------------ | --------------------------------------------- |
| ❌ Lier toutes les pages entre elles | Dilution du PageRank, confusion sémantique    |
| ❌ Utiliser des ancres vagues        | Perte de contexte sémantique pour Google      |
| ❌ Ignorer les pages orphelines      | Non-indexation, perte de visibilité           |
| ❌ Boucles ou redirections internes  | Gaspillage du budget de crawl, mauvaise UX    |
| ❌ Menus plats sans hiérarchie       | Structure de site illisible pour les robots   |

## Maillage interne et IA : Optimisation sémantique
Une stratégie de maillage interne SEO est un signal puissant pour les IA (LLM, RAG) :
- **Signal de structure** : Les liens internes aident les IA à comprendre les relations ontologiques entre les concepts de votre site.
- **Consolidation des clusters** : Un maillage dense au sein d'un cluster thématique renforce sa reconnaissance par les modèles génératifs.
- **Meilleure reformulabilité** : Les pages centrales de votre architecture, bien maillées, sont plus susceptibles d'être utilisées comme sources fiables par les IA.

## FAQ : Questions fréquentes

**Quelle est la différence entre maillage interne et netlinking ?**
Le maillage interne concerne les liens au sein d'un même site. Le netlinking concerne les liens entrants (backlinks) depuis des sites externes. Les deux sont des piliers du SEO.

**Combien de liens internes par page sont recommandés ?**
Il n'y a pas de limite fixe, mais la pertinence prime sur la quantité. Une bonne pratique est de se limiter à environ 100 liens utiles par page pour préserver le budget de crawl.

**Les ancres de lien influencent-elles le SEO ?**
Oui, massivement. Une ancre optimisée sémantiquement aide Google à comprendre le sujet de la page cible et renforce sa pertinence thématique.
